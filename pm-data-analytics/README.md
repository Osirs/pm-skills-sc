# pm-data-analytics

产品经理数据分析技能：SQL 查询生成和留存分析。分析用户数据、生成查询并识别留存模式。

## 技能 (3)

- **ab-test-analysis** — 分析具有统计显著性的 A/B 测试结果，包括样本量验证、置信区间和可操作的建议。
- **cohort-analysis** — 对用户参与度数据进行留存分析。
- **sql-queries** — 根据自然语言描述生成 SQL 查询。

## 命令 (3)

- `/pm-data-analytics:analyze-cohorts` — 对用户数据进行留存分析 —— 留存曲线、功能采用和参与趋势。
- `/pm-data-analytics:analyze-test` — 分析 A/B 测试结果 —— 统计显著性、样本量验证以及发布/延长/停止建议。
- `/pm-data-analytics:write-query` — 根据自然语言生成 SQL 查询 —— 支持 BigQuery、PostgreSQL、MySQL 等。

# pm-marketing-growth

产品营销和增长技能：营销构思、价值主张陈述、北极星指标、产品命名和定位。

## 技能 (5)

- **marketing-ideas** — 生成 5 个创意且具有成本效益的营销构思，包含渠道、消息传递和参与理由。
- **north-star-metric** — 识别一个北极星指标 (North Star Metric) 和 3-5 个输入指标 (Input Metrics)。
- **positioning-ideas** — 集思广益，提出与竞争对手不同的产品定位。
- **product-name** — 集思广益，提出 5 个独特且易于记忆的产品名称，并附带与品牌价值观和目标受众一致的理由。
- **value-prop-statements** — 根据现有的价值主张，为营销、销售和入职流程生成价值主张陈述。

## 命令 (2)

- `/pm-marketing-growth:market-product` — 集思广益提出营销构思、定位、价值主张陈述和产品名称 —— 创意营销工具箱。
- `/pm-marketing-growth:north-star` — 定义您的北极星指标和支持输入指标 —— 对业务类型进行分类并根据最佳实践进行验证。

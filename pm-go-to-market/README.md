# pm-go-to-market

产品经理进入市场 (GTM) 技能：GTM 战略、增长循环 (Growth Loops)、GTM 模式、滩头市场细分 (Beachhead Segments) 和理想客户画像 (ICP)。

## 技能 (6)

- **beachhead-segment** — 为产品发布识别第一个滩头市场细分。
- **competitive-battlecard** — 创建销售就绪的竞争对手对比卡，将您的产品与特定竞争对手进行比较。
- **growth-loops** — 识别可持续增长的增长循环（飞轮）。
- **gtm-motions** — 识别最佳的 GTM 模式和工具。
- **gtm-strategy** — 为产品发布创建进入市场战略，涵盖营销渠道、消息传递、成功指标和发布计划。
- **ideal-customer-profile** — 从研究数据中识别理想客户画像 (ICP)，包括人口统计、行为、JTBD 和需求。

## 命令 (3)

- `/pm-go-to-market:battlecard` — 创建销售就绪的竞争对手对比卡 —— 定位、功能比较、异议处理和获胜策略。
- `/pm-go-to-market:growth-strategy` — 设计可持续增长机制 —— 针对产品驱动和销售驱动战略的增长循环和 GTM 模式。
- `/pm-go-to-market:plan-launch` — 创建完整的进入市场战略 —— 滩头细分、ICP、消息传递、渠道和发布计划。
